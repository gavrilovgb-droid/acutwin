AcuTwin — Схема страниц и переходов

login.html
Авторизация
Двухшаговый вход: сначала настройка клиники, затем логин по паролю
Шаг 1: Клиника Шаг 2: Логин Демо-аккаунты ivanov / petrov / sidorov / boss
index.html
Новый приём
Форма записи пациента. Список последних 8 записей с кнопкой запуска диагностики
Форма: ФИО / возраст / пол / жалобы Последние записи (8 шт.) 🔬 Начать диагностику boss видит всех врачей
treatment.html
Тактика лечения
4 вкладки подбора рецептов. При переходе через 🔬 показывает баннер пациента
📘 Общий рецепт (677 болезней) 📙 Рецепты ТКМ 🔬 Мерид. диагностика 🌸 Подбор У-Син Баннер пациента Записать сессию Курс вручную (по дням)
history.html
История приёмов
Хронология всех сессий с фильтрами и статистикой
Фильтр: врач / пациент / дата Статистика: сессии / точки / пациенты 🔬 Начать диагностику Клик → разворачивает заметки boss видит всех врачей
atlas.html
Атлас точек
Справочник акупунктурных точек. Открывается по клику на код точки из рецепта
Поиск по коду (E36, V25…) Меридиан, описание, показания Видео / иллюстрация

Хранилища данных
💾 localStorage — постоянные данные
acutwin_clinic — название, адрес, телефон клиники
acutwin_users — список пользователей (bcrypt)
acutwin_session — текущий залогиненный пользователь
acutwin_records — все записи пациентов и сессии
acunote_{key} — заметки врача по рецепту
manual_{key} — ручной курс по дням
⚡ sessionStorage — временная передача
